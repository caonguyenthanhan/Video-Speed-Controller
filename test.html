🎬 Video Speed Controller

Một Chrome Extension mạnh mẽ và dễ sử dụng giúp bạn điều khiển tốc độ phát video trên bất kỳ trang web nào

📂 Repository GitHub:

🔗 https://github.com/caonguyenthanhan/Video-Speed-Controller

🎯 Điều khiển tốc độ linh hoạt

Từ 0.1x đến 16x với độ chính xác cao

⚡ Hiệu suất cao

Sử dụng MutationObserver để theo dõi DOM hiệu quả, không ảnh hưởng đến trải nghiệm duyệt web

🌐 Tương thích rộng

YouTube, Netflix, Vimeo và mọi trang web có video HTML5

🔄 Hỗ trợ đa video

Điều khiển nhiều video cùng lúc trên cùng một trang

🌐 Test trên các trang web thực:

Click vào các link dưới đây để test extension:

🎵 YouTube 🎬 Vimeo 📚 W3Schools 📖 MDN 🎭 Netflix 🎮 Twitch

📋 Hướng dẫn sử dụng:

  1. Cài đặt Extension: Load unpacked từ thư mục dự án
  2. Mở trang video: Truy cập YouTube, Netflix hoặc bất kỳ trang web nào có video
  3. Sử dụng Popup: Click vào icon extension trên toolbar
  4. Chọn tốc độ: Click vào các nút tốc độ nhanh hoặc sử dụng thanh trượt
  5. Phím tắt: Sử dụng Shift + (-/+/0) để điều khiển nhanh

⌨️ Phím tắt mặc định:

Shift + -

Giảm tốc độ 0.25x

Shift + +

Tăng tốc độ 0.25x

Shift + 0

Reset về tốc độ 1x

✅ Tính năng chính:

  • 🎯 8 nút tốc độ nhanh: 0.25x, 0.5x, 0.75x, 1x, 1.25x, 1.5x, 1.75x, 2x
  • 🎛️ Điều chỉnh tinh: ±0.25x với nút Fine Control
  • 📺 Hiển thị trực quan: Thông báo tốc độ trên màn hình
  • 🔄 SPA Support: Hoạt động với Single Page Applications
  • 🐛 Xử lý lỗi: Graceful error handling và user feedback
  • ⚡ Manifest V3: Tương thích với tiêu chuẩn mới nhất
